```html
    what's diffrences btw data lake and data warehouse

    1. data lake
        - stores all the raw data
        - can be petabytes (1million GBs)
        - stores all data structures
        - cost-effective
        - difficult to analyze
        - requires an up-to-date data catalog
        - used by data scientists
        - big data, real-time analytics
    2. data warehouse
        - specific data for specific use
        - relatively small
        - stores mainly structured data
        - more costly to update
        - optimized for data analysis
        - also used by data scientists and business analysts
        - ad-hoc, read-only queries
```
