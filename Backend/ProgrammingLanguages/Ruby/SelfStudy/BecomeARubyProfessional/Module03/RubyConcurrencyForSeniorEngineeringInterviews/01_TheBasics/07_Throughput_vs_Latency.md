### Throughput
~~~html
Throughput is defined as the rate of doing work or how much work gets done per unit of time.
If you are an Instagram user, you could define throughput as the number of images downloaded
by your phone or browser per unit of time.
~~~

### Latency
~~~html
Latency is defined as the time required to complete a task or produce a result.
Latency is also referred to as response time. The time it takes for a web browser
to download Instagram images from the internet is the latency for downloading the images.
~~~
