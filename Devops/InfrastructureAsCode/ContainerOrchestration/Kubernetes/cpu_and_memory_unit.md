### Memory Units
- Memory units are represented by the number of bytes.

### CPU Units
- CPU is measured in terms of, appropriately, cpu units. In Kubernetes, 1 cpu =1 vCPU/Core (for cloud providers)=1 hyperthread (for bare-metal processors).
